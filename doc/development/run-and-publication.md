# Разработка
```shell
npm run serve
```

# Сборка библиотеки
```shell
npm run build
git commit - m "commit"
git push
```

# Сборка демо проекта
```shell
npm run predeploy
```

# Публикация ветки gh-pages
```shell
npm run deploy
```

# Публикация пакета на npmjs
```shell
# (предварительно npm login)
npm publish
```
